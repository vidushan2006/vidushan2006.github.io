About

About this site — Created by VIDU

This site is a premium, private-class video platform created and maintained by VIDU. It presents curated video content (single videos and YouTube playlists) in a clean, secure interface featuring a custom player, playlist browser, and an access control layer.

Key features

  • 🎓 Private Classes hub — a grid of cards ("tiles") for each class (video or playlist). The grid and card UI are rendered dynamically from the site's data.
  • 🔒 Secure access & permissions — users must log in to view protected content; a lightweight permission table determines which accounts can open which classes. Login uses modal flows (login, access-request, access-denied).
  • 🔑 Password-unlock support — some items are additionally password-protected (a password modal appears when an item requires unlocking). Encoded passwords are validated before the viewer opens.
  • 📜 Playlist fetching from YouTube — playlist cards open a playlist popup that fetches playlist items from the YouTube Data API using the configured API key; responses are paged and cached for performance.
  • ▶️ Custom player (in-viewer) — the viewer modal mounts a custom YouTube iframe player with:
    • ▶️ play / ⏸ pause controls
    • ⏱ seek bar with progress styling
    • 🕒 duration / current time display
    • ⚙️ playback-speed menu and ⛶ fullscreen control
  • ✨ Polished UX elements — "Now playing" header, loading/progress indicators when switching content, and a playlist popup showing thumbnails and quick-play behavior.
  • ⌨️ Keyboard & accessibility — keyboard support includes:
    • Spacebar: toggle play / pause
    • ↑ / ↓ Arrow: adjust volume (visual overlay)
    • F: toggle fullscreen
  • 🗄️ Client-side caching & graceful errors — playlist results are cached to avoid redundant fetches; fetch errors are surfaced to the user with clear messages.

How to use

  1. 🖱️ Click a tile. If you're not logged in, the site will prompt you to log in or request access.
  2. 📂 If the tile is a playlist, the playlist popup will fetch and show videos — click any listed item to play it.
  3. 🔐 If the item requires a password, enter it in the unlock modal to proceed.
  4. 🎛️ Use the on-screen controls or keyboard shortcuts to control playback, seek, volume, and fullscreen.

Technical notes

  • 🔧 The playlist fetcher uses the YouTube Data API and your configured API key — ensure it is valid and has sufficient quota.
  • 📡 The custom player depends on YouTube's iframe API for playback control and playback-rate handling.
  • ♿ UI elements (modals, viewer, playlist popup, loading states) are implemented with ARIA attributes and keyboard interactions for improved accessibility.
